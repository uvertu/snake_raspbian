Создание игры "Змейка" с дистанционным управлением

Примерный план: написать "Змейку" под Raspbian OS, написать программу для управления игрой под Android. В конечном результате на экране должна ползать змейка, а пользователь управляет игрой со своего телефона, используя 4 кнопки: направление вверх, вправо, влево, вниз
